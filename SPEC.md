## 1. 問題陳述

### 1.1 目標使用者

| 族群 | 規模 | 痛點 |
|---|---|---|
| 企業 IT / 數位轉型部門 | ~5,000 | 想導入 AI 程式助理但員工英文能力不足、官方文件看不懂 |
| 中小企業主（想讓員工學 AI） | ~15 萬 | 員工抗拒新工具、學完不知道怎麼用於工作 |
| 工程師（中階想進階） | ~10 萬 | 網路教學資源多是英文、案例以美歐為主 |
| 行銷/業務/PM（非工程師） | ~20 萬 | 對 AI 好奇但不知道從哪開始 |

### 1.2 為什麼不做替代方案

- **官方 Anthropic 文件**：英文為主、無台灣在地案例
- **既有 AI 教學（中文）**：多為聊天工具教學，少程式助理
- **企業實體內訓**：每次 5-10 萬 NT$，對中小企業太貴
- **我們的解法**：中文為主 + 台灣在地案例（柚智夫妻、雷蒙三十等成功驗證模式）+ 入門/進階/企業 3 方案，NT$99-299/月

## 2. 解決方案

### 2.1 核心價值主張

> 「中文 + 台灣案例的 Claude Code 教學，讓非工程師 7 天變 AI 程式助理使用者。」

### 2.2 使用者流程

1. 選擇方案：入門 / 進階 / 企業
2. 觀看影片課程（每堂 15-30 分鐘）
3. 實作練習題（每堂 5-10 題，含台灣情境案例）
4. 提交作品 + AI 自動評分
5. 拿到結業證書 + 進入校友社群

---

## 3. 功能清單

### 3.1 MVP（必做）

- [ ] 3 方案頁面（入門 NT$99 / 進階 NT$299 / 企業 NT$29,000）
- [ ] 入門課程 20 堂（基礎對話、檔案操作、簡單自動化）
- [ ] 進階課程 30 堂（API 整合、多檔案專案、除錯技巧）
- [ ] 企業課程客製（依客戶需求）
- [ ] 練習題評分系統（執行結果比對）
- [ ] 結業證書（可下載 PDF）
- [ ] 校友社群（FB 社團 + LINE 群）

### 3.2 v2（加值）

- [ ] 一對一線上 office hour（教練）
- [ ] 企業內訓 LMS 系統
- [ ] 學習歷程記錄
- [ ] 證照認證制度

### 3.3 明確不做

- 程式語言教學（專注 Claude Code 使用，不教 Python）
- 真人實體課程（純線上）
- 國際市場（先做台灣）
- 與其他 AI 工具比較（不評論競爭對手）

## 4. 技術棧

| 層 | 選擇 | 理由 |
|---|---|---|
| 前端 | 純 HTML + Tailwind（靜態） | 課程頁 SEO 友善 |
| 影片託管 | Vimeo / YouTube unlisted | 成本最低 |
| 金流 | Stripe Checkout | 月費制自動扣款 |
| 練習題評分 | 純前端 WebContainer / Docker | 沙箱執行 |
| 證書 | jsPDF | 純前端產生 |
| 部署 | Vercel |
